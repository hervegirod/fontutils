Elements declarations

The ElementHook.supportedElements() method allows to return the declarations for the elements which are supported by the ElementHook.

Naming

The declaration is the name of the XML element in the XML format, such as for example:
  • "title" for a syntax element
  • "code" for a syntax element
  • "pre" for a syntax element
  • etc...
The method to get the declaration for an element is StyledElement.getDeclaration(). Elements which have no declaration return "none" as their declaration by default.
