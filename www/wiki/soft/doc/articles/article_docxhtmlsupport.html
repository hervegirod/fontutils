Docx hand-written html articles support

    1  Supported elements
    2  div elements
    3  a elements
    4  See also

There is a limited level of support for the Hand-written html articles in the DOCX generation.

Supported elements

The following elements are supported:
  • "hi" elements: h<x> tag
  • "img" elements
  • "a" elements
  • "ul", "ol," and "li" elements
  • "div" elements
  • "p" elements
Note: "nav" elements are skipped.

div elements

The border around "div" elements is be taken into account.

a elements

The whole text for "a" elements is taken into account, even if the link contains several portions of text (for example with "span" elements as children).

However, if the "a" element contains at least one "img" element, no hyperlink will be included in the resulting document, but the first found image will be added instead.
