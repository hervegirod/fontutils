Colors

Colors can be specified using either:
  • An HTML color name (for example red)[1]
    See htmlcolorcodes.com/color-names/ for the list of named HTML colors
  • A color specified using A RGB or RGBA pattern (For example #00FF00 for a Green color)

Notes
