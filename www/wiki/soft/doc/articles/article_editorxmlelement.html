List of XML elements in the Editor

The XML elements in the editor are the elements which can be edited. The list of these elements is:
  • The articles and the index article
  • The Resource definition files
  • The Image definition files
  • The Infobox definition files
  • The Dictionary
  • The Categories
  • The menus
  • The footer and header files
