It is possible to show an auto-complete Search box at the top of the articles in the help GUI. This will allow to search for articles or even tfor terms in the whole wiki (see search for more information).

You can also search for a term in any page by typing Ctrl+F as you could do in any Web browser, see help search in page for more information.

Search options

There are the same options for the help search box, as there are for the website search box:
  • By default the Search box allow to search for the articles titles and their table of content sub-titles
  • There can be a Search box for the articles titles only
  • Or there can be no Search box


Full text search

Additionnally, depending on the value of the fullTextSearch property, the help search box can also include a full text search option.
