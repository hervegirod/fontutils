Custom search specification

    1  Grammar
    2  Content
    3  Example
    4  See also

The custom search categories are specified through the "searchCustomSpec" property which point to an XML file defining the categories which can be used for the search.

Grammar

See The custom Search categories Schema.

Content

The file contains several category elements specifying the categories which will be handled in the search box. For example:
   <customSearch>
      <category name="major" />
      <category name="minor" />   
   </customSearch>

Example

For example:
      search=custom
      searchCustomSpec=customSearch.xml
