Comparison with Markdown syntax

DocJGenerator uses an XML-based syntax. However the concepts handled by the Markdown markup exist in DocJGenerator. If you really want to use a Markdown syntax, you can define an article in Markdown markup (see markdown syntax).

Comparison table

Concept Markdown syntax docJGenerator XML syntax
Titles series of "#" characters at the beginning of a line (example: ## title for a level 2 title) title element
Lists series of "*" characters at the beginning of a line (example: * item 1) ul or ol element
Blockquote "> text" structure blockquote element
Interwiki links "[other article name](other article path)" structure ref element
External link "[external link text](external link path)" structure a element
Bold or italic text enclosing _ or ** characters b or i element
Code presentation Enclosing "`" characters code element
Pre presentation Enclosing "```" lines pre element
Image "![<image caption>](<image path>)" structure img element
