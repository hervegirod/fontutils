Dictionnary

The wiki generator automatically generates a dictionnary of all the articles. This dictionnary can be accessed from any page in the wiki.

Adding a description to the dictionnary

It is possible to add a content to the dictionnary. This can be specified in an XML files with the "dictionnary" root.
For example: 
      <dictionnary>
         The wiki has ${count} articles.
      </dictionnary>
