This article presents a list of tutorials.

The solutions of the tutorials are in the docGenerator-tutorial-<version>.zip

Wiki generation tutorials

The first tutorial explains how to create your first wiki, and explains notions about:
  • The index
  • Creating an article, both in XML or using the Mediawiki markup
  • Adding an internal wikilink
  • Adding categories to an article
  • Adding footer and header files to the wiki
  • Customizing the left menu
  • Generating the wiki
The second tutorial explains how to:
  • Add images and use these images in articles
  • Use a resource file and link to this file in articles
The header tutorial explains how to customize the header.

The StyleSheet theme tutorial explains how to:
  • Use a custom StyleSheet theme for the wiki
The hand written articles tutorial explains how to add hand-written articles in the wiki.

Wiki editor tutorials

The editor tutorial explains how to use an editor to edit the wiki.

Packages tutorials

The packages tutorial explains how to use packages to separate Open Source and Closed Source content.

The second packages tutorial explains how to use packages to separate wiki content which is finished from content which is still unfinished.

Review system tutorial

The Review system tutorial explains how to add a review system in the wiki.

API documentation elements tutorials

The API documentation elements basic tutorial explains how to reference APIs in the wiki:
  • Add links to existing APIs
  • Embed APIs in the wiki

API tutorials

The API basic tutorial explains how to use the generator in another Java application.

Help system tutorials

The Help system Swing tutorial explains how to:
  • Generate a zip file containing the Help content
  • Use this content to create a help system for a Swing application
The Help system JavaFX tutorial explains how to:
  • Use the same help content to create a help system for a JavaFX application
The Help system Swing themes tutorial explains how to:
  • Generate a zip file containing the Help content with StyleSheet themes
  • Use this content to create a help system for a Swing application allowing to switch themes at runtime
The Context-sensitive Help Swing tutorial explains how to:
  • Add context-sensitive help to a Swing application
The Context-sensitive Help JavaFX tutorial explains how to:
  • Add context-sensitive help to a JavaFX application

Document generation tutorials

The Document generation tutorial explains how to:
  • generate a PDF or DOCX document
