The left panel shows the tree of the Editor window shows the wiki roots.

Structure of the tree

The tree is structured as the wiki file system. Elements in the tree are shown corresponding to their type:
  • : Root directories
  • : Packages
  • : Sub-directories
  • : doc-files directories
  • : resource files
  • : Image files
  • : The dictionary file
  • : The categories files
  • : articles
  • : The index article
  • Image definition files
  • Resource definition files
  • Infobox definition files
